# TriAngels Exit Node

Стандарты, автоматизация и эксплуатация exit-нод TriAngels Mesh (Headscale + Tailscale)

---

## 🧭 О проекте

Этот репозиторий содержит полный стандарт развертывания и эксплуатации exit-ноды в сети TriAngels Mesh.

Цель — обеспечить:

- единый подход к развёртыванию VPS
- стабильную и предсказуемую работу exit-нод
- масштабируемость инфраструктуры
- минимизацию ручных ошибок

---

## 🚀 Что входит

- 📄 Стандарт развертывания exit-ноды
- ⚡ Network Tuning (BBR, fq, sysctl)
- 🔁 Настройка NAT и маршрутизации
- 🧪 Проверки (netcheck, MTU, статус)
- 📘 Паспорт сервера
- 🤖 Скрипты автоматической установки

---

## 🏗 Архитектура

TriAngels Mesh построен на:

- Headscale (control plane)
- Tailscale (mesh overlay)
- Exit Nodes (выход в интернет)
- ACL (разграничение доступа)
