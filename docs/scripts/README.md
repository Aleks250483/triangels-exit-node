# Скрипты TriAngels Exit Node

В этой директории находятся скрипты автоматизации:

- установка exit-ноды
- проверка состояния (verify)
- сетевой тюнинг
- вспомогательные утилиты

---

## План

- triangels-exit-installer.sh — основной установщик
- triangels-exit-verify.sh — проверка (gold state)
- triangels-exit-hardening.sh — безопасность (в будущем)
